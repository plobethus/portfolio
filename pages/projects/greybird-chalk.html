Greybird‑Chalk Dark

Greybird‑Chalk is a dark GTK/Xfwm4 theme that fuses the beloved Greybird layout with the cosy pastel hues of the Chalkboard VS Code palette. It delivers a unified look across GTK 3 apps, XFCE panels and window decorations—all while keeping Greybird’s trademark legibility and small‑details polish.

The inspiration for this project was the visual studio theme Chalkboard by McKenzie Bottoms. After I installed this theme and fell in love with it in visual studio, I decided my terminal should look the same. After creating a terminal scheme for the XFCE terminal, my next target was the xfce desktop itself. I was already using Greybird-dark, so by editing the Greybird repository I was able to create Greybird-Chalk.

P.S. - I also styled this website with the Chalkboard color scheme.

Overview

Built on top of the greybird upstream sources, this repository:

  • Switches to a pastel‑on‑charcoal colour scheme (Charleston Green base).
  • Ships gtk‑3.0, gtk‑2.0, xfwm4, Metacity, Openbox and Plank assets for full desktop coverage.
  • Adds a one‑click Greybird‑Chalk‑Dark metatheme so Appearance managers pick it up cleanly.
  • Keeps Greybird’s compact paddings, subtle shadows and accessibility tweaks.

Palette at a glance

{
  "base"   : "#1F2D2D", // Charleston Green
  "blue"   : "#BDEAFF", // Diamond
  "yellow" : "#FFFCA4", // Canary
  "green"  : "#C8FF9B", // Mint Green
  "pink"   : "#FFB0B0"  // Melon
}

The colours above map onto $base_color, selection tints, info bars and semantic feedback (error / warning / success).

Installation

  1. git clone https://github.com/henrymoran/greybird‑chalk.git
  2. meson setup builddir --prefix="$HOME/.local"
  3. meson compile -C builddir && meson install -C builddir
  4. Select Greybird‑Chalk‑Dark in Settings ▸ Appearance.
