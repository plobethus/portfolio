My Portfolio Website

This website serves as a personal showcase of my software development work, writing, and research. It's a fully hand-coded site that prioritizes performance, security, and maintainability across the full stack.

Overview

This project was built entirely from scratch using standard web technologies — HTML, CSS, JavaScript, and PHP — without any frontend frameworks. It’s hosted on an Apache server running RHEL and backed by a MariaDB database for secure message storage via the contact form. I also implemented Cloudflare to provide HTTPS, CDN caching, and basic DDoS protection. Another thing I included is prism.js for syntax highlighting in my project pages.

Beyond the portfolio, this server also powers the UHDL Discord List API, a JSON API that serves approved, non-expiring Discord invites for UH Computer Science courses.

My goal was to create a fast, accessible, and responsive site that I fully control — every part of the stack is open and transparent.

Hosting

This website is completely self hosted

This form securely submits user messages to a backend PHP script with CSRF protection and server-side validation. Messages are stored in MariaDB.

Stack & Security Features

  • Frontend: HTML5, CSS (no frameworks), vanilla JavaScript
  • Backend: PHP 8 with best-practice session/cookie security
  • Database: MariaDB for storing contact messages
  • Web server: Apache on RHEL with HTTPS via Cloudflare
  • Security: CSRF tokens, input sanitization, secure session management
  • Services: UHDL Discord List API (DB-backed JSON at /api/servers.php)
